# PSPM CLI Guide

PSPM (Prompt Skill Package Manager) is a CLI tool for managing prompt skills across AI coding agents.

## Full Documentation

For the complete CLI reference, see: **[docs.anyt.io/pspm/cli-reference](https://docs.anyt.io/pspm/cli-reference)**

## Quick Start

```bash
npm install -g @anytio/pspm
pspm install          # Install skills from lockfile
pspm add <skill>      # Add a skill
pspm list             # List installed skills
pspm publish          # Publish a skill
```
